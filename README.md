# Appt - Appointment Reminder Bot

## Description
Appt is a simple Python-based appointment reminder bot.
It allows users to enter their appointment date and time and reminds them at the correct time.

## Features
- Add appointment
- Automatic reminder
- Simple console interface

## Technologies Used
- Python
- datetime module
- time module

## How to Run
1. Install Python
2. Run: python appt.py
3. Enter your appointment details

## Future Improvements
- Email reminders
- SMS reminders
- Multiple appointments support
